# 微信表情包创建工具

一个简单易用的微信表情包处理工具，帮助用户快速处理表情包并生成头像和图标。

## 主要功能

### 1. 表情包处理
- 自动分割多个表情的图片
- 支持透明背景和纯色背景图片处理
- 智能识别和分离表情元素
- 支持合并和删除表情元素
- 自动调整尺寸至微信表情规格

### 2. 头像和图标生成
- 从表情包生成微信头像（240×240像素）
- 生成表情面板图标（50×50像素）
- 确保符合微信官方规范
- 一键下载功能

## 使用说明

1. **处理表情包**
   - 上传包含多个表情的 PNG 或 JPG 图片
   - 系统自动分割和处理表情
   - 可以编辑、合并或删除表情元素
   - 下载处理好的表情包

2. **制作头像和图标**
   - 从处理好的表情中选择合适的图片
   - 自动生成头像和图标
   - 检查是否符合规范要求
   - 下载生成的头像和图标


## 技术栈

- Next.js
- React
- TypeScript
- Tailwind CSS
- Canvas API

## 开发环境设置

```bash

# 运行开发服务器
npm run dev

# 构建生产版本
npm run build
```
